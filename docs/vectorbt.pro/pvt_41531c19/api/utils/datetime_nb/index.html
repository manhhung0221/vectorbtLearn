    datetime_nb

    datetime_nb module¶

    Module providing Numba-compiled utilities for efficient date and time computations.

    DTCS named tuple¶

    Datetime component status enumeration.

    Status codes returned by within_fixed_dtc_nb and within_periodic_dtc_nb.

    DTCST(
    SU=-3,
    EU=-2,
    U=-1,
    O=0,
    I=1
)

    Fields

    SU
    Start matched; remaining components are unspecified.
    EU
    End matched; remaining components are unspecified.
    U
    Unknown status.
    O
    Outside.
    I
    Inside.

    Args

    Named tuple with datetime components, where missing values are indicated by -1.
    other_dtc : DTCNT
    Datetime components to match against.

    Returns

    bool
    True if the components match, False otherwise.
